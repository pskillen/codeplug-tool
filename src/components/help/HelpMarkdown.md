# HelpMarkdown

## Purpose

Renders long-form help hub content from markdown strings in the content manifest. Internal links starting with `/` use React Router.

## Props

| Prop      | Type     | Notes         |
| --------- | -------- | ------------- |
| `content` | `string` | Markdown body |
